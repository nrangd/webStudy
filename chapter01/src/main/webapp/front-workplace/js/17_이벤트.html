# HTML의 이벤트 속성

  • 앞에 on이 붙는 HTML속성들은 자바스크립트 이벤트 속성이다 (onclick, onmouseover, onkeydown, onfocus, onmouseleave ...)
  • 해당 이벤트 속성에 콜백을 달아두면 이벤트 정보를 담고있는 이벤트 객체가 넘어오면서 이벤트가 발생하게 된다
  • 해당 요소의 HTML속성에는 자바스크립트 문법을 값으로 사용할 수 있다
  • 또는 해당 요소를 자바스크립트에서 선택하여 addEventListener() 메서드를 통해 해당 이벤트에 대한 콜백 함수를 추가해 둘 수 있다

# 이벤트 위임

  • 기본적으로 부모 요소에 추가한 이벤트는 그 위에 놓여있는 자식 요소에서도 발동한다
  • 이벤트를 발생시킨 객체의 인스턴스가 이벤트 객체의 target에 실려온다

# 이벤트 버블링

  • 중첩된 요소에서 이벤트가 발생했을 때 해당 이벤트가 자식 요소에서부터 부모 요소까지 전이되는 현상
  • target과 currentTarget을 사용해 이벤트 버블링 도중에 발생시키고 싶은 이벤트만 골라낼 수 있다
  • 혹은 e.stopPropagation() 을 통해 이벤트 전파를 방지 할수 있는듯 하다 다만 이렇게 사용할 시 문제가 생길 여지가 있으므로 가능한 위의 방법을 쓰자 출처 - https://inpa.tistory.com/entry/JS-JS-📚-버블링-캡쳐링

# 이벤트 캡쳐링

  • 버블링보다 먼저 일어나는 이벤트 처리 단계
  • 버블링과는 반대로 부모 요소부터 자식 요소까지 전이되는 이벤트 처리 순서
  • 기본적으로는 비활성화 되어있다
  • 캡처링이 모두 진행되고 난 후에는 원래대로 버블링이 진행된다

# 요소의 기본동작 막기

  • 어떤 요소들은 이벤트 발생전에 실행되는 기본 동작들이 있다
  • 원하는 이벤트를 처리하기 위해 기본동작을 막아야 하는 경우들이 있다
