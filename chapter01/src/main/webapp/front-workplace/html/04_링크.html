# a태그 사용하기

- a 태그의 href속성에 원하는 url을 적어서 사용

- 자세한 내용이 궁금하다면여기를 클릭하세요
- a 태그를 누르면 웹 브라우저의 주소창에 href 속성에 적힌 값을 입력하는 효과가 있다
- 구글 가기(google.com)
- 구글 가기2(./google.com)
- 구글 가기3(../google.com)
- 구글가기4(/google.com)
- 구글가기5(//google.com)
- 구글가기6(http://google.com)

# 웹에서 사용할 수 있는 상대 경로의 종류 (./, ../, /, //)

- https://comic.naver.com/webtoon/detail?titleId=821793&no=88&week=mon를 기준으로
- href="/naver.com"은 https://comic.naver.com/naver.com을 의미한다(최상위 디렉토리를 기준)
- href="naver.com"은 https://comic.naver.com/webtoon/naver.com을 의미한다(현재 디렉토리를 기준)
- href="./naver.com"은 https://comic.naver.com/webtoon/naver.com을 의미한다
- href="../naver.com"은 https://comic.naver.com/naver.com을 의미한다
- href="//naver.com은" https://naver.com을 의미한다
