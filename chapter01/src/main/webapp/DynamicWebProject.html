1. Eclipse EE install

  • 웹 개발자용 버전으로 설치
  • File - New를 했을 때 Dynamic Web Project가 있다
  • 오른쪽 위에 Perspective를 누르면 자바 개발자용으로 돌아갈 수도 있다

2. Apachi Tomcat 설치 (Web Application Server, WAS)

  • 웹 서버(Web Server)는 기본적으로 정적인 자원(이미지, 웹페이지, 텍스트 등...)만 응답해주는 프로그램을 의미한다
  • 동적(상황에 따라 달라져야하는)인 웹 페이지를 서비스 하기 위해 필요한 것이 Web Application Server라고 할 수 있다
  • WAS는 프로그래밍 언어로 만들어진 서버 페이지를 응답할 수 있는 기능이 추가되어 있다
  • 톰캣 9.0이상 버전은 오라클 JDK와 호환되지 않는다
  • Eclipse의 Servers에 등록놓으면 이클립스가 알아서 관리해준다
  • 설치된 곳의 server.xml을 수정하여 여러 서버 설정들을 변경할 수 있다(ex. 포트번호)

3. 새 프로젝트 생성하기

  • File - New - Dynamic Web Project로 생성
  • Target Runtime이 ervers에 등록한 서버로 설정되어 있어야 한다
  • 바로 생성하지 말고 Next 두번 눌러서 Generate web.xml 체크 후 생성하기 (체크 후 생성시 webapp/WEB-INF/web.xml이 생성됨)

4. 생성한 프로젝트를 서버에 등록 후 접속하기

  • Servers에 등록한 서버를 우클릭 후 Add and Remove로 프로젝트를 등록한다
  • 서버 우클릭 후 Start (서버 실행)
  • 서버가 실행되어 있는 동안에는 웹 브라우저로 서버에 접속하여 웹 페이지를 볼 수 있다 (접속 경로: http://내IP주소:포트번호/프로젝트명/해당파일의webapp내부경로)
  • Ctrl + F11도 가능하다 (서버를 알아서 재시작 한 후에 웹브라우저로 접속을 해준다)
